# 분석
    비밀번호를 해시하기 때문에 nodejs 서버의 cpu에서 병목 현상이 나타났다.
    이에 반해 nginx 서버와 db 서버 둘 다 매우 여유로웠다.
    처리량을 늘리기 위해서는 nodejs 서버 인스턴스를 늘리는 수 밖에 없어 보인다.
